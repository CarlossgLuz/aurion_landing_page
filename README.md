## Descrição

Este é um projeto de _Front-end_ que simula a página de lançamento ```(Landing Page)``` de um produto fictício, o pré-treino **"Ignite"** da marca **"Aurion Nutrition"**.

O design é moderno, responsivo e focado em converter o usuário, levando-o a se cadastrar na "Lista VIP". O projeto implementa diversas técnicas de JavaScript para melhorar a experiência do usuário (UX) e a usabilidade, desde animações de scroll até um formulário interativo.

## Funcionalidades

* **Design Responsivo:** Totalmente adaptável para desktops, tablets e celulares.
  
* **Menu Hamburger:** Navegação mobile funcional com animação (```script.js```).
  
* **Scroll Suave:** Navegação fluida entre as seções da página.

* **Animações na Rolagem:** Elementos (cards, formulário) aparecem suavemente usando IntersectionObserver.

* **Formulário Interativo:** Validação de campos (nome, e-mail) e feedback visual.

* **Persistência Local:** O formulário de _"Lista VIP"_ salva os leads no localStorage do navegador.

* **Efeito de Iluminação:** O produto na seção "Hero" reage sutilmente ao movimento do mouse (```script.js```).

* **Botão "Voltar ao Topo":** Aparece após o usuário rolar a página.

* **Acessibilidade:** Implementação de link "Pular para conteúdo" (```skip-link```).

* **Easter Egg:** Contém um "Modo IGNITE ULTRA" secreto ativado pelo [Código Konami](https://pt.wikipedia.org/wiki/C%C3%B3digo_Konami) (```script.js```).

---

## Tecnologias

* **HTML5:** Estrutura semântica (```index.html```).

* **CSS3:** Estilização avançada com Flexbox, Grid e animações customizadas (```style.css```).

* **JavaScript (ES6+)**: Manipulação do DOM, eventos, IntersectionObserver e localStorage (```script.js```).

* **Nenhuma dependência**: Projeto construído com "vanilla" JS, HTML e CSS puros.
